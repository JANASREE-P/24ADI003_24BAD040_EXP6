Scenario 1: Diabetes Prediction (Bagging)
Dataset:
https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database
About Dataset:
The Pima Indians Diabetes Dataset is used to predict whether a patient has diabetes based on medical attributes like glucose level, BMI, age, and blood pressure. It is widely used for binary classification problems in healthcare.
Scenario 2: Customer Churn Prediction (Boosting)
Dataset:
https://www.kaggle.com/datasets/blastchar/telco-customer-churn
About Dataset:
This dataset contains customer details from a telecom company. It helps predict whether a customer will churn (leave the service) based on factors like tenure, monthly charges, and contract type.
Scenario 3: Income Prediction (Random Forest)
Dataset:
https://www.kaggle.com/datasets/uciml/adult-census-income
About Dataset:
The Adult Income Dataset predicts whether a person earns more than $50K per year using demographic and employment-related features such as age, education, occupation, and working hours.
Scenario 4: Heart Disease Prediction (Stacking)
Dataset:
https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset
About Dataset:
This dataset is used to predict the presence of heart disease based on medical parameters like cholesterol, heart rate, and age. It is commonly used in classification and healthcare analytics.
Scenario 5: Credit Card Fraud Detection (SMOTE)
dataset:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
About Dataset:
This dataset contains transactions made by credit cards, where fraud cases are highly imbalanced compared to normal transactions. It is ideal for demonstrating class imbalance handling using techniques like SMOTE.
